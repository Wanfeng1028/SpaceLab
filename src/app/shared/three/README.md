# Three.js Adapted Visualizations

This folder contains Three.js visualization components adapted from open-source projects for the SpaceLab homepage.

## Sources

- **GlobeStream3D**: https://github.com/hululuuuuu/GlobeStream3D
  - Adapted for the Earth Flyline (second screen) visualization
  - Core logic: `globe-stream/` directory

- **earth-flyLine**: https://github.com/notbucai/earth-flyLine
  - Reference for flyline animation techniques

- **three.js examples**: https://github.com/mrdoob/three.js
  - Bloom post-processing: `webgl_postprocessing_unreal_bloom.html`
  - Points and sprites: `webgl_points_sprites.html`, `webgl_buffergeometry_points.html`
  - Adapted for the Blue Moon Tree (third screen) visualization
