## 使用说明

### 访问地址
- 登录: http://localhost:4200/login
- 注册: http://localhost:4200/register
- 管理后台: http://localhost:4200/admin

### 创建管理员账号
1. 访问 http://localhost:4200/register 注册账号
2. 通过数据库将角色改为 admin：
   ```sql
   UPDATE users SET role = 'admin' WHERE email = 'your@email.com';
   ```

### 功能说明
- 登录后自动跳转管理后台
- 管理后台显示文章统计
- 可以新建、编辑、删除、发布文章
- 文章编辑支持 Markdown 和预览
- 未登录访问管理页面会自动跳转登录页
